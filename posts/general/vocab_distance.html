Means and median distance between lemma occurances

Fletcher Hardison

I was reading a Koine Greek text last night and I realized that I was able to get a handle on certain new voccabulary because it was frequent within the section I was reading. I have no idea if it would be a frequent word within the work as a while, but the distance between encounters with the word in that section was not great. This got me wondering if this idea of distance would be helpful as a metric for vocabulary ordering or for creating reading orders.

So I wrote some code using the gnt_data from vocab-toolsJames Tauber’s work.. The code is available in this gist.

The two tables below show the results 25th-50th rows of the output sorted by mean and median respectively. Looking at the output, it seems that the mean distance follows overall frequency (total) which makes sense. But is not the case for the median distnace which generally follows, but not quite as closely.

Notice that sorted by median, does not follow frequency at all. The total number of occurance and mean vary widely. The lower the median, the higher the probably that a word occurs multiple times close together. So we might expect a word like θηρίον to be rare within the corpus, but in certain sections it might be common.

In the end is this data particulary useful? Probably not. It could be helpful as a starting point to see if if a word tends to cluster. If so, then it should be possoble to use vocab-tools to find which pericopes, paragraphs, chapters etc. where the word clusters. This could be helpful in finding readings so that we can use a familar corpus such as the GNT to teach words that might be rare within that corpus while being more common within the wider Greek corpus. Of course this could all be hogwash. Hopefully, I can write the code to find pericopes where the words cluster.
